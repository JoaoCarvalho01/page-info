Informações

Use este painel para marcar horários de entrada, consulta rápida de registros e deixar notas sobre cada marcação. Você pode inserir o nome da pessoa e escolher um horário (ou deixar em branco para usar a hora atual de Brasília).

Notas

Esta página funciona offline no seu navegador — os registros não são persistidos entre recarregamentos. Se quiser salvar, posso gerar a versão que grava em arquivo (.csv) ou envia para um backend/Firebase.
